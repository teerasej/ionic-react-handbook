
# กำหนด App Icon ใน iOS 

1. เปิดโปรเจคขึ้นมาใน Xcode 

```bash
npx cap open iOS 
```

2. จาก sidebar ด้านซ้าย, เลือก App → Assets.xcassets → AppIcon

3. ค่อยๆ ลากไฟล์รูปภาพที่ได้จากเว็บ ไปวางให้ตรงช่อง
