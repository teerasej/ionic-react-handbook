
# กำหนด Splash Screen ใน iOS 

1. เปิดโปรเจคขึ้นมาใน Xcode 

```bash
npx cap open iOS 
```

2. จาก sidebar ด้านซ้าย, เลือก App → Assets.xcassets → Splash

3. ค่อยๆ ลากไฟล์รูปภาพขนาด 4096 x 4096 pixel ไปวางให้ตรงช่อง
